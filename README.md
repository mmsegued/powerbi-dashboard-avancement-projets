# Dashboard Power BI – Avancement de projets

## Contexte
Projet réalisé dans le cadre du parcours Data Analyst (OpenClassrooms).

## Objectif
Construire un tableau de bord permettant de suivre l’avancement d’un portefeuille de projets (KPI, filtres, synthèse).

## Données & préparation
- Import des données dans Power BI
- Nettoyage et enrichissement via Power Query
- Mise en place de champs utiles (statuts, délais, indicateurs)

## KPI (exemples)
- Nombre de projets
- Répartition par statut
- Avancement moyen
- Projets en retard / à risque

## Livrables
- Captures du dashboard (dossier `screenshots/`)
- (Option) fichier PBIX (si taille 

## Compétences démontrées
Power BI (modélisation, dataviz), Power Query (préparation), storytelling, restitution.
